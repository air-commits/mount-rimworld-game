# 图形界面快速开始指南

## 快速开始

### 1. 安装依赖

```bash
pip install pygame
```

或者安装所有依赖：

```bash
pip install -r requirements.txt
```

### 2. 运行游戏

```bash
python game_gui.py
```

## 基本操作

### 移动
- **W/↑**: 向上移动
- **S/↓**: 向下移动
- **A/←**: 向左移动
- **D/→**: 向右移动

### 交互
- **鼠标左键点击NPC**: 开始对话
- **输入消息 + 回车**: 发送对话

### 快捷键
- **[M]**: 游戏菜单
- **[Q]**: 任务列表
- **[C]**: 基地管理
- **[I]**: 背包（开发中）
- **[ESC]**: 返回/关闭

## 游戏界面说明

### 顶部HUD
显示玩家信息：
- 姓名和等级
- 生命值
- 金币
- 当前位置

### 世界视图
- 彩色地图（不同地形类型）
- 玩家（蓝色圆点）
- NPC（绿色圆点）
- 实时渲染

### 菜单系统
- 方向键选择
- 回车确认
- ESC返回

## 常见问题

**Q: 游戏窗口打不开？**
A: 确保已安装pygame：`pip install pygame`

**Q: 按键无响应？**
A: 确保游戏窗口处于焦点状态，检查输入法

**Q: 游戏运行慢？**
A: 关闭其他程序，降低分辨率
